자동화 QA 기록 템플릿 (API + UI 기본 스모크)

## 실행 환경
- 실행일시: `$(date)`
- 백엔드: `KIFU_API_URL` (기본 `http://127.0.0.1:8080`)
- 프론트엔드: `KIFU_FRONTEND_URL` (기본 `http://127.0.0.1:5173`)

## 실행 명령
- `bash scripts/full-qa.sh [API_URL] [FRONTEND_URL]`
- 사용자 세션 데이터 점검(샘플 계정 필요):
  - 기본(전체 출력): `KIFU_AUDIT_EMAIL=<email> KIFU_AUDIT_PASSWORD=<password> python3 scripts/kifu_state_audit.py --api http://127.0.0.1:8080`
  - 콘솔 정리 모드: `KIFU_AUDIT_EMAIL=<email> KIFU_AUDIT_PASSWORD=<password> python3 scripts/kifu_state_audit.py --api http://127.0.0.1:8080 --summary`
  - 원본 저장: `KIFU_AUDIT_EMAIL=<email> KIFU_AUDIT_PASSWORD=<password> python3 scripts/kifu_state_audit.py --api http://127.0.0.1:8080 --save /tmp/kifu-audit.json`

### 사용자 데이터 정합성 수집 (디버그)

- 위의 API 경로 스냅샷을 한 번에 확인하려면 `kifu_state_audit.py`를 실행하세요.
- 기본은 `=== KIFU_AUDIT_START ===`/`=== KIFU_AUDIT_END ===` 구간의 JSON이며,  
  정리 모드에서는 `=== KIFU_AUDIT_SUMMARY ===`로 축약된 핵심 지표만 출력됩니다.
- 동일 증상 재현시 이 결과를 붙여주시면 `/trades`, `/portfolio`, `/review`, `/bubbles` 간 동기화 이슈 추적이 빨라집니다.
- 너무 길 때는 아래처럼 “발췌 모드”로 보세요.
  - 전체 저장: `KIFU_AUDIT_EMAIL=<email> KIFU_AUDIT_PASSWORD=<password> python3 scripts/kifu_state_audit.py --api http://127.0.0.1:8080 --save /tmp/kifu-audit.json`
  - 즉시 핵심만 추출: `python3 scripts/kifu_audit_extract.py /tmp/kifu-audit.json`
  - 또는 1단계만으로 요약 실행: `KIFU_AUDIT_EMAIL=<email> KIFU_AUDIT_PASSWORD=<password> python3 scripts/kifu_state_audit.py --api http://127.0.0.1:8080 --summary --save /tmp/kifu-audit.json`
  - 추출기 출력에는 `=== KIFU_AUDIT_BRIEF === ... === KIFU_AUDIT_BRIEF_END ===` 형식으로 핵심 수치가 나옵니다.
